# 🚀 muslim


## 🧩 Features
- 🌙 Custom splash screen
- 🧭 Smooth navigation between screens using Bottom Navigation Bar
- 🏁 Using `initialRoute` for controlled app navigation
- 📿 Custom Sebha screen
- 📖 Custom Quran screen
- 📜 Custom Hadith screen
- 📂 Reads text files from assets
